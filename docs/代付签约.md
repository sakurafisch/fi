# 代付签约

流程：柜面录入->对公经理审批->本级领导审批->主管盖章->柜面桌面待办

## 资料

1. XX银行代收代付业务申请表（盖公章）

2. 代付业务协议（盖公章 + 私章）（本级正职领导签名）

3. 完税证明（税种须包括基本养老保险等，证明劳务关系）或劳动合同

4. 一套账户资料

5. ?? 单位签约代付协议免费申请（对公经理自拟，上级分管领导签名）

> 一套账户资料的定义
>
> - 营业执照
>
> - 开户许可证
>
> - 法人身份证
>
> - 非法人《授权书》
>
> - 经办人身份证

## 先验印三份

- XX银行代收代付业务申请表

- 代付业务协议

- 非法人《授权书》

## 柜面录入注意事项

以下内容必填，即使系统不控制必填

- 开户行行号

- 业务类型：必勾批量代付

提交后打A4单，流程流转至对公经理审批，此时代付限额仍待集中录入

此时盖章材料【代付业务协议】应该物理传递至主管，但未能盖章

## 柜面桌面待办

- 逐项扫描

- 打印A4，此时应有生效的代付限额信息
