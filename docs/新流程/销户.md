# 销户

## 销户预约提醒

- “人行征询”亮显 => 基本户

- “人行征询”隐藏 => 非基本户

当“人行征询”按钮亮显时，必点。

- 查看账户信息（注意是否有余额，可能需要出售结算业务申请书）

- 查看签约情况（注意是否签约代发，可能要先撤约再销户）

- 查看预申请信息（销户原因/经办人/预约时间）

## 销户综合受理

### 先变更后销户

若客户信息已经变更，应先完成账户信息、预留印鉴变更再销户

### 已工商注销、单位印章已销毁或遗失

参见【2024】100号（20240123） || 2024第十期知识特辑（20240419）

- [国家企业信用信息公示系统](https://shiming.gsxt.gov.cn/)（打印作为附件）

- 出具《情况说明》

> 出具《情况说明》分两种情况
>
> - 法定代表人系实际控制人 => 法定代表人出具《情况说明》
>
> - 法定代表人并非实际控制人 => 二选一：要么股东共同出具情况说明，要么实际控制人（而不是法定代表人）出具情况说明

### 查询持有凭证

内部管理->查询->登记簿->客户服务->客户凭证

### 附件检查

> 在【票据补扫】时，需要扫描全套的账户资料。即使 [销户资料接收](#销户资料接收) 仅要求扫描《销户申请书》。

1. 《销户申请书》

2. 《销户业务申请说明函》（【2025】82号 20250120）

3. 开户许可证《基本存款账户信息》（注意：收回原件）

4. 《准予注销登记T知书》

5. 《印鉴卡》客户联（或遗失声明）

6. 《清税证明》

7. 法人身份证（和手机号）

8. 经办人身份证（和手机号）

### 前置准备

- 撤约（如代付协议）

- [出售结算业务申请书](../凭证出售.md)（用于销户后续余额转出）

- 验印

### 验印

- 销户业务申请说明函

### 交易路径

客户服务->账户->开销户->销户综合受理（线上）

输入三选一：“申请编号”、“单位名称”、“客户账号”

选中需销户的账户，点击“确认受理”按钮。

### 销户资料接收

- 联动检查账户状态是否为“正常”

- 联动“账号销户协议检查”

- 核对无误后点“确认”按钮，打印通凭

- 扫描《销户申请书》

### 流程跟踪

内部管理->流程跟踪->销户流程跟踪

### 后续处理

- 网点审批

- 费用收取

- 人行销户核准/备案

- 销户余额转出

- 正式销户

- 票据补扫

> 转出金额 = 账户余额 + 已计提利息 - 手续费

### 不可交票

- 开户许可证《基本存款账户信息》

- 《印鉴卡》客户联（或遗失声明）
