# 开户

## 公司带来的材料

1. 《营业执照》

2. 开户许可证《基本存款账户信息》

3. 公章

4. 私章（法人名章）

5. 财务章

6. 法人身份证（和手机号）

7. 经办人身份证（和手机号）

8. 非法人《授权书》

9. 大于百分之二十五的股东身份证一比一尺寸复印件

10. 有限公司还需要提供《章程》

11. 印章备案表

12. 经营地的用地证明（当经营地和注册地不在同一地方时需要提供）

## 公司打印的

1. 开立申请书

2. 授权书

3. 协议

4. CRS

5. 签约确认书

6. 网银签署授权书

7. 印鉴申请书

## 一般户额外柜面打印

开户申请（内容大概：因公司业务需要，申请开立壹个一般户...）
